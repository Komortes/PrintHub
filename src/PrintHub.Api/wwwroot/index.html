PrintHub

Проверка сервиса

Локальный HTTP сервис для управления принтерами и печатью PDF. Подходит для сайта, ERP, CRM и внутренних интеграций, где веб-система должна печатать на локальном компьютере пользователя.

PrintHub сам скачает PDF по указанному URL и подготовит его к печати.

Если принтер не выбран, PrintHub отправит задание на системный принтер по умолчанию.
